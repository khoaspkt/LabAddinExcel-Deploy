Công cụ hỗ trợ lập dự toán trên Excel: tra cứu công việc, chèn định mức, phân tích đơn giá, cập nhật dữ liệu và đồng bộ bộ đơn giá.

Tải và cài đặt Add-in Mở file VSTO
Sau khi bấm cài đặt, trình duyệt có thể tải file setup.exe về máy. Hãy mở file vừa tải để bắt đầu cài đặt. Nếu Windows hoặc trình duyệt hỏi xác nhận, chọn tiếp tục nếu bạn tin tưởng nguồn cài đặt.
Cài đặt nhanh
Dùng file setup.exe để cài add-in vào Excel.
Tự cập nhật
Add-in được publish bằng ClickOnce/VSTO, có thể cập nhật khi có bản mới.
Dữ liệu riêng
Dữ liệu định mức, đơn giá được cập nhật riêng bằng chức năng trong add-in.

Hướng dẫn cài đặt

  1. Bấm nút Tải và cài đặt Add-in.
  2. Mở file setup.exe vừa tải về.
  3. Làm theo hướng dẫn cài đặt trên màn hình.
  4. Mở Microsoft Excel.
  5. Kiểm tra tab LAB trên Ribbon Excel.
  6. Nếu chưa có dữ liệu, bấm nút Cập nhật dữ liệu trong add-in.

Nếu máy báo thiếu thành phần cài đặt, hãy kiểm tra Microsoft Office, .NET Framework và Visual Studio Tools for Office Runtime.
