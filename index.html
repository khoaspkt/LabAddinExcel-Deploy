Excel VSTO Add-in

Công cụ hỗ trợ lập dự toán trực tiếp trên Excel

LabAddinExcel hỗ trợ tra cứu công việc, chèn định mức, phân tích đơn giá, cập nhật dữ liệu và đồng bộ bộ đơn giá ngay trong Microsoft Excel.

⬇ Tải bộ cài đặt
Tải file nén, giải nén rồi chạy file .bat để cài đặt add-in.
Cài đặt đơn giản
Người dùng chỉ cần tải file, giải nén và chạy file cài đặt nhanh.
Dễ cập nhật
Add-in và dữ liệu có thể được cập nhật khi có phiên bản mới.
Dùng trong Excel
Sau khi cài đặt, tab LAB sẽ xuất hiện trên Ribbon Excel.
✓ Sẵn sàng tải về
File cài đặt được đóng gói dạng .zip để tải xuống thuận tiện hơn trên trình duyệt.
Tên file tải về
Install_Lab_AddinExcel.bat.zip
1
Máy tính dùng hệ điều hành Windows.
2
Đã cài Microsoft Excel/Office phiên bản phù hợp.
3
Cho phép chạy file cài đặt nội bộ do đơn vị cung cấp.

Hướng dẫn cài đặt

Tải bộ cài đặt
Bấm nút Tải bộ cài đặt ở phía trên để tải file về máy.
Giải nén file tải về
Nhấp phải vào file Install_Lab_AddinExcel.bat.zip và chọn giải nén.
Chạy file cài đặt
Mở file .bat vừa giải nén và làm theo hướng dẫn trên màn hình.
Mở Excel và kiểm tra
Mở Microsoft Excel, kiểm tra tab LAB trên Ribbon và cập nhật dữ liệu nếu cần.

Lưu ý khi cài đặt

Nếu Windows hoặc trình duyệt hiển thị cảnh báo bảo mật, hãy kiểm tra đúng file do đơn vị cung cấp rồi chọn cho phép chạy. Không tải file cài đặt từ nguồn không rõ ràng.
Không thấy tab LAB?
Đóng toàn bộ cửa sổ Excel, chạy lại file cài đặt, sau đó mở lại Excel để kiểm tra.
Thiếu dữ liệu?
Vào tab LAB trong Excel và bấm chức năng Cập nhật dữ liệu.
Cài đặt không thành công?
Kiểm tra Microsoft Office, .NET Framework và Visual Studio Tools for Office Runtime trên máy.
